# PlaywrightFundamentalsSG

A small Playwright test project used for learning Playwright fundamentals and running example tests.

Prerequisites
-
- Node.js (14+)

Quick start
-
- Install dependencies: `npm install`
- Run tests: `npx playwright test`
- Generate and view HTML report: `npx playwright show-report`

Where results live
-
- HTML report: `playwright-report/`
- Raw test artifacts: `test-results/`

Notes
-
- Tests are located in the `tests/` folder. See `tests/01_Basics/Lab209.spec.ts` for an example.
